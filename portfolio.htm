My Projects

boldHER

Girls Who Code Summer Immersion Program

boldHER is a women empowerment website that aims to provide resources and a platform for women to support each other by asking and answering questions.

Softwares/Languages Used: HTML, CSS, JavaScript, Django Framework, MapBox API

No Stress Dress

PixelHacks II Hackathon: Best Presentation

No Stress Dress is a mobile application that allows users to not only have an app pick out their outfits, but also plan out outfits for the week, plan outfits for travel, wear rarely worn clothes more often, and online shop while adding fantasy items to a virtual closet.

Softwares/Languages Used: HTML, CSS, DropSource, Sketch

Botlucky

Superposition II Hackathon: Best .tech Domain

Botlucky is a website that combats food waste by allowing users to share food with people nearby. The website includes a google form embedded in map that pinpoints the the location of where the person who is sharing food is.

Softwares/Languages Used: HTML, CSS, GoogleMaps API, Photoshop

California Wildfires

California Wildfires is a WebMap that reflects upon three major wildfires in California in 2018 and presents a visual map of wildfire threat throughout California.

Softwares/Languages Used: ArcGIS Online (Geographic Information System software)
